Learning Note

Linear Regression

线性回归

​ 线性回归：给定数据集，\({D=\{(x_1,y_1),(x_2,y_2),...(x_i,y_i)\}}\) ，其中 \[ x_{i} =(x_{i1};x_{i2};...;x_{in}),y_i \in R; \] 线性回归试图学得一个线性模型，以尽可能准确地预测实值输出。 \[ f(x_i) = wx_i+b;使得f(x_i)\cong y_i \]

最小二乘法

​ 如何确定\(w,b\) 的值，关键在于如何衡量\(f(x_i)和y_i\) 的差值。均方误差是线性回归中最常用的性能度量，因此可以试图让均方误差最小。均方误差对应常用的欧式距离(Euclidean distance)。基于均方误差最小化来对模型进行求解的方法称为最小二乘法（Least square method）。 \[ \begin{align} & (w^*,b^*) = arg min\sum_{i=1}^{n}(f(x_i)-y_i)^2\\ \end{align} \]

\(a\cdot b=\sum_{i=1}^{n}a_ib_i\) %累加和
